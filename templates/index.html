welcome to colesurfs

i designed this dashboard for staying on top of the swell forecast for the northeast

it is exactly the data i use for forecasting, consolidated into one interface

claude helped with the code, you can review it on github

for my main site, visit coleheine.com

tips
  • tap on a buoy name to reveal associated spot data for that buoy
  • tap on the name of a spot to link to Surfline's live camera
  • compare the GFS and Euro models to get a sense of how reliable the forecast is: model agreement = more reliable forecast
  • this site works best as a web app: tap 3 dot menu, then share, then add to homescreen
