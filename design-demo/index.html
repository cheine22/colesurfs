A single small chip next to the LIVE indicator, driven by the new backend _status metadata and the snapshot age. Absent when everything is fresh — it only appears when there's something worth knowing. On mobile it docks in the same place, text shortened to the dot + one word.

3 · OUTAGE MODAL — PALETTE ALIGNMENT

The outage modal is the only surface with hardcoded amber/brown hex values outside the Hue Mac palette. The proposed version keeps the amber warning accent (via --amber) but sits on standard surfaces, so it reads as part of the app and stays correct in light mode for free.
